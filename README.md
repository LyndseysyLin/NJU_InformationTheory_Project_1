# Huffman Coding Compression System

本项目实现了一个基于霍夫曼编码（Huffman Coding）的无损压缩系统。通过模块化设计，实现了从数据压缩、二进制序列化到性能分析的完整闭环。

## 1. 项目结构
本项目采用模块化设计，清晰划分了算法层、IO层、分析层及测试层：

```text
InformationTheory_Project_1/
├── tests/                 # 测试套件
│   ├── test_integration.py # 系统集成测试
│   └── test_unit_huffman.py # 算法核心单元测试
├── analysis.py            # 信息论指标计算与性能分析
├── huffman_codec.py       # 霍夫曼编码核心算法
├── io_handler.py          # 二进制文件读写与序列化处理
├── main.py                # 项目主入口
├── README.md              # 项目文档
└── source.txt             # 待压缩的原始数据

```

## 2. 核心模块职责

- **huffman_codec.py**: 核心算法实现，包含霍夫曼树的构建与编解码逻辑。
- **io_handler.py**: 处理二进制文件读写，将霍夫曼码表与压缩后的二进制流封装为 `.bin` 文件，确保数据无损存取。
- **analysis.py**: 信息论评估工具，计算信源信息熵、实际平均码长及编码效率，用于定量评价压缩性能。
- **main.py**: 项目主入口，集成上述模块完成自动化的压缩-解压校验流程。
- **tests/**: 包含单元测试与集成测试，确保算法正确性及系统兼容性。

## 3. 性能评估指标

本系统集成了一套严谨的性能分析工具，程序运行时会自动输出以下关键指标，用以验证编码效率：

- **理论极限**: 基于香农熵的理论压缩极限（bits/symbol）。
- **实际平均码长**: 衡量算法的压缩性能（bits/symbol）。
- **编码效率**: 编码结果对理论极限的逼近程度。
- **压缩比**: 直观的体积优化指标。
